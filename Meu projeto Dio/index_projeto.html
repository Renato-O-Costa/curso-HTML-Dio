1. Origem da Web

A Web foi criada por Tim Berners-Lee em 1989, enquanto trabalhava no CERN, com o objetivo de compartilhar informações científicas.

2. Web 1.0

A Web 1.0 consistia em páginas estáticas, com pouca ou nenhuma interatividade.

  • Somente leitura
  • Sem login ou formulários
  • Designs simples

3. Web 2.0

A Web tornou-se mais interativa e colaborativa. Usuários passaram a gerar conteúdo.

  1. Redes sociais
  2. Blogs e fóruns
  3. Streaming e vídeos

4. Web 3.0

Conhecida como Web Semântica, visa tornar a Web mais inteligente com dados conectados e descentralizados.

Inclui conceitos como blockchain e IA.
