Introdução

Voltar ⬆
Por mais longa que seja a caminhada o mais importante é dar o primeiro passo.
(Autor: desconhecido)

Toda jornada se inicia no primeiro passo, sendo esse o marco decisivo para quem almeja avançar em qualquer área na vida, e nesse quesito, o bootcamp de Front-end da Dio/Santander, nos auxilia bem.

Já na introdução, somos apresentados à estrutura básica do HTML, que é a linguagem de marcação usada para estruturar as páginas na Web. Juntamente com isso, nos é mostrado ferramentas que são extremamente úteis para o aprendizado, como o inspetor de elementos do Chrome e o editor de código VSCode, que nos permite usar várias extensões que auxiliam na codificação.

Na apresentação das tags, que são as partes que indicam o tipo de elemento que elas representam, ficamos sabendo que algumas delas são tags contêiners e outras não (as chamadas tags vazias). Outrossim, as tags possuem atributos que nos permitem realizar mudanças e formatações nas tags onde os atributos são usados.

Como todas as páginas utilizam-se de textos, saber formatá-los é essencial na construção de nossos sites. E como isso não bastasse, aprendemos sobre como inserir listas ordenadas e não ordenadas, bem com criar links que possibilitam acessar outras páginas de conteúdo similar e complementar. Enfim, aprender HTML [2] faz-me sentir como um construtor de bases, para páginas na web.

Formulários HTML

Voltar ⬆

Os formulários são essenciais para a interação do usuário com o site, permitindo que ele envie informações, como comentários, cadastros e pesquisas. Aprender a criar formulários HTML é fundamental para qualquer desenvolvedor web.

Estruturando o HTML

Voltar ⬆

Estruturar o HTML corretamente é crucial para garantir que as páginas sejam bem organizadas e fáceis de entender. Isso inclui o uso adequado de tags, hierarquia e semântica. Durante o curso, também aprendemos que alguns elementos como H2O e CO2 podem ser apresentados em HTML com o uso da tag <sub>, permitindo representar fórmulas químicas corretamente na página.

Utilizando mídias no HTML

Voltar ⬆

Incorporar imagens, vídeos e outros tipos de mídia enriquece a experiência do usuário e torna o conteúdo mais atraente. Aprender a usar as tags apropriadas para mídias é uma parte importante do desenvolvimento web.

Criando tabelas no HTML

Voltar ⬆

As tabelas são úteis para organizar dados de forma clara e estruturada. Aprender a criar e estilizar tabelas no HTML é essencial para apresentar informações tabulares de maneira eficaz.

HTML Semântico
Voltar ⬆

O HTML semântico é uma prática recomendada que melhora a acessibilidade e a compreensão do conteúdo. Utilizar tags semânticas ajuda os mecanismos de busca a entender melhor o contexto das informações apresentadas.

Conclusão
Voltar ⬆

Concluindo, o curso de HTML 1 da Dio/Santander [3] é uma excelente introdução para quem deseja aprender a criar páginas web. Com uma base sólida em HTML, os alunos estão bem preparados para avançar para tecnologias mais complexas e desenvolver sites dinâmicos e interativos.
